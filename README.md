# SB Gemmological Solutions Ltd - Landing Page

A modern, responsive landing page for SB Gemmological Solutions Ltd, built with pure HTML, CSS, and JavaScript.

## Features

- **Hero Carousel**: Three rotating cover images with navigation controls
- **Responsive Design**: Fully responsive layout that works on all devices
- **Expandable Sections**: Interactive "What We Offer" sections with expandable content
- **Smooth Scrolling**: Smooth navigation between page sections
- **Modern UI**: Clean, professional design matching Figma specifications

## Setup

1. Clone the repository
2. Ensure all images are placed in the `assets/` folder with the correct filenames
3. Open `index.html` in a web browser
